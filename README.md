# Labora Café Website

A modern, responsive website for **Labora Café**, designed to showcase the brand, menu, locations, and customer experience.

This project is a complete rebuild of the original Framer website using a modern web development stack to improve performance, scalability, and long-term maintainability.

## Features

* Responsive design across desktop, tablet, and mobile devices
* Interactive menu browsing
* Smooth page transitions and animations
* Optimized images and assets
* Accessibility-conscious interface
* SEO-friendly structure
* Fast loading performance

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Git & GitHub

## Project Structure

```text
src/
├── assets/        # Images, icons, and static assets
├── components/    # Reusable UI components
├── pages/         # Website pages
├── layouts/       # Shared layouts
├── hooks/         # Custom React hooks
├── utils/         # Helper functions
└── styles/        # Global styles
```

## Getting Started

Clone the repository:

```bash
git clone https://github.com/kqngyn/labora-website.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
